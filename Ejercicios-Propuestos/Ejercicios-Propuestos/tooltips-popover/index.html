Tooltips

Los tooltips son cuadros de texto que aparecen al hacer hover sobre un elemento

Son componentes opt-in, es decir, debemos de inicializarlos nosotros, la manera más fácil es hacerlo es seleccionando todos los elementos de la página con el valor de data-bs-toggle = tooltip.

Código que debemos incluir en la sección script de nuestro html para habilitarlos:

            
    const tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'));
    const tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
        return new bootstrap.Tooltip(tooltipTriggerEl);
    });

Opciones: posicionamiento

Podemos controlar la posición usando el atributo data-bs-placement (top,right,bottom,left). Ejemplos:

Ejemplo de codigo para el primer boton

                
    <button type="button" class="btn btn-secondary" data-bs-toggle="tooltip"
            data-bs-placement="top" data-bs-title="Tooltip arriba">
            Tooltip arriba
    </button>

Popover

Los popover son parecidos a los tooltip, pero son cuadros más grandes, los usaremos cuando tengamos que mostrar más texto, un texto con título o un enlace.

En cuanto a su comportamiento son similares, pero típicamente en vez de ser activados con un hover y desaparecer al mover el ratón, los popover suelen mantener abiertos hasta que el usuario interactúa para cerrarlos.

También son elementos opt-in por lo que deberemos de activarlos al igual que los tooltip. Codigo que debemos incluir en la sección script de nuestro html para habilitarlos:

            
    const popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'));
    const popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
        return new bootstrap.Popover(popoverTriggerEl);
    });

Ejemplo de popover: por defecto estará a la derecha y tendremos que cerrarlo haciendo click otra vez en el botón

Código para el popover simple en un botón

            
<button type="button" class="btn btn-secondary" data-bs-toggle="popover" data-bs-content="Popover">
Popover
</button>

Opciones: posicionamiento

Al igual que los tooltips, podremos cambiar la posición con el atributo data-bs-placement(top,bottom, left,right)

Opciones:titulo

Podemos añadir un título con el atributo data-bs-title

Código para el popover con título

                
    <button type="button" class="btn btn-lg btn-danger" data-bs-toggle="popover" data-bs-title="titulo popover"
            data-bs-content="contenido popover, puede ser mas largo que los tooltips" >
            Popover con título
    </button>

Opciones: cerrar al hacer click en otro lado

Podemos hacer que el popover se cierre al hacer click en otro lado añadiendo data-bs-trigger="focus" a los atributos del botón

Código para el popover con título

                
    <button type="button" class="btn btn-lg btn-danger" data-bs-toggle="popover" data-bs-title="titulo popover"
            data-bs-content="contenido popover, puede ser mas largo que los tooltips" data-bs-trigger="focus">
            Popover con título
    </button>

Opciones: activar al hacer hover

Podemos hacer que el popover active con hover añadiendo data-bs-trigger="hover" a los atributos del botón

Código para el popover que se activa con hover

                
    <button type="button" class="btn btn-lg btn-danger" data-bs-toggle="popover" data-bs-title="titulo popover"
                    data-bs-content="contenido popover, puede ser mas largo que los tooltips" data-bs-trigger="hover">
                Popover que se activa al hacer hover
                    </button>
